# Budget App (Android)

Приложение для учета расходов на строительство дома.

## Что умеет
- Справочник категорий расходов (расширяемый): `materials`, `works`.
- Подкатегории для расходов: кровля, фундамент, стены, земельные работы, дренаж, наружная отделка, внутренняя отделка, отопление, водоснабжение, электричество, сеть, кухня, санузлы.
- Учет источников финансирования (счета): дебетовая карта, кредитная карта, наличные и любые пользовательские.
- Хранение баланса по каждому счету.
- Внесение расходов (с указанием счета, категории, подкатегории).
- Внесение доходов (пополнение счета).
- Планирование будущих доходов и расходов.
- Расчет прогнозного бюджета на заданную дату.

## Технологии
- Kotlin
- Jetpack Compose
- Room (SQLite) — локальная БД в приложении
- Architecture Components (ViewModel + Flow)

## Запуск локально
1. Откройте проект в Android Studio (Giraffe+).
2. Дождитесь синхронизации Gradle.
3. Запустите `app` модуль на эмуляторе или Android устройстве.

## Как получить APK через GitHub Actions
1. Запушьте изменения в репозиторий.
2. Откройте вкладку **Actions**.
3. Запустите workflow **Build debug APK** вручную (Run workflow) или дождитесь автозапуска на `main/master/work`.
4. После завершения job скачайте артефакт **budget-app-debug-apk**.
5. Установите `app-debug.apk` на телефон.

> На телефоне может понадобиться разрешить установку из неизвестных источников для браузера/файлового менеджера, через который запускается APK.

## База данных
Используется локальная Room/SQLite БД на устройстве: `budget_app.db`.
Общий сервер не требуется на текущем этапе.

## Структура
- `app/src/main/java/com/ovk/budgetapp/data` — Room сущности, DAO, база.
- `app/src/main/java/com/ovk/budgetapp/ui` — экран и ViewModel.
